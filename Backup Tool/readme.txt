**********BALATRO BACKUP/RESTORE TOOL**********


Requirement:
Ensure Python is installed and added to PATH.


How to use RESTORE TOOL:
1. Connect android device to laptop/pc. Ensure USB debugging is turned on.

2. Run "balatro_restore_saves.bat" file to start restoring saves.

3. Copy your backup files (that you want to restore) to the "to_restore" folder. Ensure you only have one meta.jkr and profile.jkr file each in there.

4. This script will overwrite the selected Android profile with your backup. Read the instructions carefully!
