**********BALATRO SAVE TRANSFER TOOL**********
		(Android to PC)


Requirement:
Ensure Python is installed and added to PATH.


How to use:
1. Connect android device to laptop/pc and run the .bat file. Ensure USB debugging is turned on.

2. Give backup permission from your android device when prompted.

3. Ensure you have an internet connection to install dependencies. 

4. This script will create two backup files: 1-meta.jkr and 1-profile.jkr in the SAME directory. If backup is successful but files are not found, please check the newly created "extracted backup" folder for the files.
