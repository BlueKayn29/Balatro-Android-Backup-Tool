**********BALATRO SAVE TRANSFER TOOL**********
		(PC to Android)

WARNING: You will be asked to select a PC profile to copy and an Android profile to overwrite
This will REPLACE the selected Android profile with the PC profile
It is HIGHLY RECOMMENDED to BACKUP your Android profile before proceeding

Requirement:
1. Ensure Python is installed and added to PATH.
How to download and/or add Python to Path: https://www.howtogeek.com/197947/how-to-install-python-on-windows/

2. Android Balatro APK is debuggable
How to make an app debuggable: https://gist.github.com/amahdy/87041554f62e384bee5766a958fd4f9a


How to use:
1. Connect android device to laptop/pc and run the .bat file. Ensure USB debugging is turned on.

2. Ensure you have an internet connection to install dependencies. 

3. Read the warnings and instructions carefully. You may lose an important save if not careful
